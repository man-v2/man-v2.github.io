Python多环境安装配置

2018-12-03 python 0 评论 字数统计: 360(字) 阅读时长: 1(分)

python多环境安装配置

一、下载安装

1. 下载地址

  • python27下载地址(https://www.python.org/ftp/python/2.7.15/python-2.7.15.amd64.msi)
  • python37下载地址,下载Windows x86-64 embeddable zip file(https://www.python.org/ftp/python/3.7.1/python-3.7.1-embed-amd64.zip)

2. 安装顺序

  • python27自动安装
    执行安装程序进行安装
    安装完后，在cmd输入python -V, 并显示 “Python 2.7.15”,则成功。
    安装程序会默认把python27的an安装目录配置到系统变量
  • python37 进行手动安装
    解压python37,复制程序路径。在我的电脑-属性-高级系统变量-环境变量-高级系统设计，会弹出环境设置的窗口。
    在下面的系统变量窗口，找到”path”,点击编辑，把python37路径添加到最后。win10直接新增添加。 win7在添加前，输入;来分割路径。
    eg:   ;c:\python27;c\python37
  • 同时把scripts目录也添加到环境变量
    eg:   ;c:\python27\scripts;c\python37\scirpts

二、配置

  • 修改python3文件名称
    先修改python.exe为python3.exe,其他再对照是否需要i需改

    1. python3.dll

    2. python3.exe

    3. python37.dll

    4. python37_d.dll

    5. python37_d.pdb

    6. python3_d.dll

三、测试

  • 测试python27
    python -V, 打印：Python 2.7.15，则成功
  • 测试python37
    python3 -V, 打印：Python 3.7.15，则成功

四、结束
